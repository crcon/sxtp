# 山西独立储能二次调频收益测算平台

基于《山西电力二次调频辅助服务市场交易实施细则》(2025) 的独立储能收益测算与展示平台。

## 快速预览

```bash
cd webapp
python3 -m http.server 8088
# 打开 http://localhost:8088
```

或直接访问 GitHub Pages：https://crcon.github.io/sxtp/webapp/

## 功能模块

- **收益测算**：结果总览、小时/日/月度收益明细
- **运行分析**：调频电量、SOC 曲线、性能指标雷达
- **财务分析**：全生命周期现金流、投资回收期、IRR/NPV、敏感性分析
- **基础数据**：电价与补偿标准、设备参数、政策与计算方法

## 核心计算口径

**山西现行单里程制**：
```
调频收益 = 实际调节深度 D^R × 性能折算 K结算 × 申报价 b
K结算 = β · λ · K实际 ,  λ = 2 / max(K实际)  (α=0.5, β=0.8)
```

**独立储能调频量价补偿**：
```
C_es = 调频时段充电电费 × η − 调频时段放电电费 （负值不补）
```

**扩展双部制估值**（分析性工具，非山西现行结算）：
```
R_ext = Q_cap·h·P_cap + D·K·P_mile + C_es − C_deg − OC
```
